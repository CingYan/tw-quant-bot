## 🛠️ 技術細節

### 架構設計

```
BacktestEngine
├── Portfolio（資金管理）
│   ├── Position（持倉）
│   └── Order（訂單）
├── Strategy（策略邏輯）
│   ├── on_bar()      # 每根 K 線回調
│   ├── buy/sell()    # 交易接口
│   └── record_signal() # 訊號記錄
└── PerformanceMetrics（績效評估）
    ├── 報酬率計算
    ├── Sharpe Ratio
    ├── Max Drawdown
    └── 交易統計
```

### 關鍵技術

**1. 訂單執行流程**
```python
1. 策略生成訊號 → 創建 Order
2. 檢查資金/持倉是否足夠
3. 計算手續費 + 證交稅
4. 更新現金 + 持倉
5. 記錄訂單
```

**2. 倉位計算**
```python
# 支援零股交易
position_value = total_value * percentage
shares = int(position_value / price)
```

**3. 績效指標**
```python
# Sharpe Ratio（年化）
sharpe = (avg_return / std_dev) * sqrt(252)

# Max Drawdown
max_dd = max((peak - equity) / peak)
```
